# Obsidian Literature Note

The default `[Item] Obsidian Literature Note` template is intended as the first practical note format for Zotero-to-Obsidian workflows.

It is designed to:

- preserve Zotero metadata in YAML frontmatter
- expose translation fields when available
- stay readable in plain Markdown
- remain easy to customize later in the template editor

Suggested workflow:

1. Select a Zotero item.
2. Create an item note from `[Item] Obsidian Literature Note`.
3. Export or sync the note as Markdown.
4. Open the note in Obsidian and continue editing there.
